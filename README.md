# napcat-plugin-github-sub

NapCat GitHub 订阅推送插件 - 监控仓库 Commits / Issues / Pull Requests 并渲染图片推送到群

## 功能特性

- 📦 **仓库订阅** - 订阅任意 GitHub 公开/私有仓库
- 🔔 **事件推送** - Commits、Issues、Pull Requests、评论自动推送
- 🖼️ **图片渲染** - 通过 Puppeteer 渲染精美推送图片，含 diff 代码高亮
- 🎨 **主题切换** - 白天/黑夜/自定义主题配色
- ✏️ **自定义模板** - 支持自定义 HTML 模板渲染推送图片
- 🔑 **多 Token 轮转** - 支持多个 GitHub Token 轮转使用，提高 API 速率限制
- 👑 **权限管理** - 主人系统 + 可配置是否允许普通成员订阅
- 🎛️ **WebUI 控制台** - 可视化配置面板，管理订阅/Token/主题/模板

## 指令列表

默认指令前缀为 `gh`。

| 指令 | 说明 |
|------|------|
| `gh 帮助` | 显示帮助 |
| `gh 列表` | 查看当前群订阅 |
| `gh 全部` | 查看所有订阅 |
| `gh 订阅 <owner/repo>` | 添加订阅（默认监控全部类型） |
| `gh 取消 <owner/repo>` | 取消订阅 |
| `gh 开启 <owner/repo>` | 开启订阅 |
| `gh 关闭 <owner/repo>` | 关闭订阅 |

> 监控类型等细节配置请前往 WebUI 控制台管理

## WebUI 配置面板

插件内置可视化配置面板，支持：

- 仪表盘总览（订阅数/推送群/轮询状态/GitHub 连通性测试）
- 基础配置（Token 管理/API 地址/轮询间隔/调试模式）
- 权限设置（主人 QQ/允许普通成员订阅）
- 订阅管理（添加/编辑/删除/开关订阅，群选择器）
- 渲染主题（白天/黑夜/自定义配色）
- 自定义 HTML 模板（Commits/Issues/PRs，支持预览）
- 调试日志（实时查看/筛选/清除）
- 深色模式自适应
